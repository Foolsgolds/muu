# Muu

Windows 11 向けのグリッド型ランチャー。霧 (む) → muu。

## 特徴

- **5×5 グリッド** にアプリ・フォルダ・コマンドを登録して素早く起動
- **グローバルホットキー** `Win+Ctrl+Alt+M` でマウスカーソル位置にポップアップ
- **コマンドパレット型検索** — アプリ・ファイル・電卓 (`=2+3`)・Web 検索
- **キーボード操作** — ↑↓←→ で移動、Enter で起動、Tab で巡回
- **中央セルでドラッグ** してウィンドウを移動
- 半透明＋影付きのフローティング UI

## インストール

### winget

```powershell
winget install foolsgold.Muu
```

### 手動インストール

[Releases](https://github.com/yanqirenshi/muu/releases) から `Muu.exe` をダウンロードして任意のフォルダに配置するだけ。

## 使い方

1. 起動するとシステムトレイに常駐
2. **`Win+Ctrl+Alt+M`** でマウスカーソル位置にランチャー表示
3. **左クリック** でアプリ起動、**右クリック** でセル設定
4. 検索バーに入力すれば、アプリ／ファイル／Web／計算結果から素早くアクセス

## 必要環境

- Windows 10 / 11 (x64)
- 単独 exe — .NET ランタイム不要
